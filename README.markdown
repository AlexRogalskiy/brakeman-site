# Brakeman Site

This is the code for building https://brakemanscanner.org

## Development

The site is built using [Jekyll](https://jekyllrb.com/).

Changes to the site content should be made in `source/`. Markdown is preferred.

To build/test:

    cd source
    bundle install
    jekyll serve
